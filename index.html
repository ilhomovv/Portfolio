REACT MEETUPS APP
React React Hooks CSS HTML Javascript Firebase Responsive

About this project:

This is a Single-Page-Application made with React, where you can save your past and present Meetups (exhibitions, movies, parties, online events...) you have attended or plan to attend. You can save new Meetups, consult all your Meetups, save a Meetup as Favorite and update the Meetup if necessary.

Database: Firestore of Firebase to store the Meetups in a NoSQL document-oriented database

  • React-DOM
  • React-router
  • Components
  • UseEffect Hook
  • UseContext hook
  • UseState hook
  • useHistory hook
  • React build-in "Card" and "Link" Components
